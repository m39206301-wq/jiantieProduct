## 仓库简介

本仓库收集中式生活礼俗中各类请柬、邀请函、讣告的经典中文文案，覆盖婚、寿、生、迁、业、学、节、丧九大场景。每则文案均标注【篇幅】【适用场合】【风格】（多数附【适配载体】），可直接检索套用，也可作为改写底本。写作前先读《00 请柬格式与载体指南》，可系统掌握称谓、敬语、具名、日期写法与单页／双开页／三折页的文案分配方法。

## 使用方法

### 四步选用：场景 → 风格 → 篇幅 → 载体

1. **定场景**：按宴请性质进入对应分册（01–08），先定场合色调——喜（红金热烈）、寿（红金敬祝）、节庆家宴（暖、团圆）、白事（白黑克制）。
2. **选风格**：在分册内按分组挑选风格，如传统中式／古风文言／现代文艺／酒店正式／电子口语；注意全篇语体统一，文言套语不与白话混用。
3. **定篇幅**：按渠道与信息量选档（见下表）；微信短信用短档，纸质正帖正文用中档，含流程回执的盛典用长档。
4. **配载体**：按印制形式分配文案——单页放短档、双开页“封面短句＋内页中档正文”、三折页“封面一句、扉页一段、内页一柬、副页一表、封底一名”。具体分配规则与示例见《00 请柬格式与载体指南》第五章。

### 占位符说明

- 文案中 **××× / ××** 为占位符，使用时须替换为真实的**姓名、日期、地点、电话**等信息。
- 替换姓名用全名，不用乳名、昵称、缩写；两人姓名之间用“暨／和”连接。
- 替换日期时请**公历农历并书**、核对星期与干支（查万年历），农历用汉字数字（廿、卅、初、冬月、腊月）。
