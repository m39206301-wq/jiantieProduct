# 升学宴 · 谢师宴 · 成人礼 · 同学会 —— 中文请柬文案集（终稿）

## 场景总述

本文集覆盖四类人生礼俗场景，共收录文案 **37 则**，其中长文案（完整全文）11 则。

- **升学宴**（金榜题名：高考 / 考研 / 留学，13 则）：主角是学子，具名发帖的是父母（或全家）；核心词为“金榜题名、蟾宫折桂、鹏程万里、鱼跃龙门”；情绪为“分享喜悦＋答谢亲朋”。
- **谢师宴**（毕业谢师，9 则）：主客是老师，发帖人是学生（个人 / 班级 / 届全体）；核心在敬语——敬备薄酒、恭请光临、恭候台光、拨冗赏光；幽默版常见于同班学生邀请各科老师。
- **成人礼**（传统冠礼 / 笄礼 ＋ 现代 18 岁成人礼，8 则）：传统仪式用语出自《仪礼·士冠礼》《礼记》，字字有出处；现代成人礼多由学校组织（跨成人门、戴成人帽、宣誓），家庭宴请则仿升学宴体例。
- **同学聚会**（周年聚会、毕业 N 年，7 则）：发帖人是“同学会筹备组 / 筹委会”；情绪为怀旧＋召唤；经典收束句“别因你的缺席而使聚会失色”。

每则文案统一标注【篇幅】【适用场合】【风格】【适配载体】；正文中 ××× / ×× 为占位符，使用时替换为实际姓名、日期、地点。

## 附录：写作规范与术语表

### 附一、格式与措辞规范（精简）

**1. 请柬通用格式（五部分）**
- 标题：封面（或首行居中）写“请柬 / 请帖 / 邀请函”。
- 称谓：顶格写被邀请人姓名全称＋尊称，后加冒号；不写绰号、昵称；夫妇合请写“××暨××”或“××夫妇”，不用顿号逗号分隔两人姓名。
- 正文：写清事由、时间（公历＋农历＋星期，年月日时俱全）、地点（酒店名＋厅号＋路牌，必要时附乘车路线）。
- 结尾敬语：恭请光临 / 敬请光临 / 恭候台光。传统竖版讲究“提行顶格”——“恭请”另起，“光临”顶格；己方谦词（敬备、恭候、谨订）不顶格。
- 落款：具名＋敬词（敬邀 / 谨邀 / 敬约）＋发柬日期；发柬时间以宴前 3 天至 2 周为宜。

**2. 升学宴具名惯例（重点）**
- 由父母具名，学子不具名：邀请人写父母姓名（“××、××敬邀”或“××夫妇敬邀”），学子以“为爱子 / 爱女 / 小女××举办升学宴”出现于正文——谁做东谁具名。
- 称谓谦辞：称自家孩子用“爱子、爱女、小女、小儿、犬子（旧式）、学子××”；称对方孩子用“令郎、令爱”。
- 传统竖版写法：由右至左、由上至下；“谨订于公历×年×月×日／农历×年×月×日（星期×）／为××升学庆典／是日假座××酒店×厅／敬备薄酌恭候／光临／×时恭候，×时入席／××敬约”。租借场地用“假座”，自家场地用“在 / 于”。
- 书写工具：传统讲究用黑色钢笔或毛笔，不用铅笔、圆珠笔。
- 谦辞惯例：宴席再丰盛也须谦称“薄酌、菲酌、薄酒、便宴、粗馔”。
- 电子请柬补充信息：公交 / 自驾路线、停车场、联系人与电话、“因诸事繁多未能登门邀请，敬请海涵”是常见得体补笔。

**3. 谢师宴敬语体系（重点）**
- 对老师（尊长）：用“恭候台光”“拨冗赏光”“敬治薄酒”“略备薄酒 / 菲酌”“恭请光临”“如蒙赏光，不胜荣幸”；落款“××届全体学生敬邀”“学生××敬约”。
- 对同门 / 同学（平辈）：可用“诚心期盼您的莅临”“恭候光临”，语气可亲切如“许久不见，近来可好”。
- 禁忌：请老师不可用命令式时间限定（“务必出席”失礼）；不宜在请柬中提礼金礼品；“谢师”主题应落在“感谢教诲”，避免夸饰排场。
- 名篇用语：敬治薄酒，恭候台光；诚望百忙之中拨冗赏光，吾辈荣幸之至。

**4. 成人礼措辞注意**
- 传统仪式：冠礼祝辞引用《仪礼》须准确（“令月吉日，始加元服。弃尔幼志，顺尔成德”），“元服”指头服（冠），勿误写“原服”；三加顺序为缁布冠→皮弁→爵弁（笄礼为发笄→发钗→钗冠）。
- 年龄代称：男子二十曰“弱冠”，女子十五许嫁而笄曰“及笄”；现代法定成年为 18 岁，行现代成人礼时用“十八而志”，不宜再称“弱冠 / 及笄之年”（那是 20 / 15 岁），引用需注明古今之别。
- 现代仪式：学校版邀请函须写清流程（跨成人门、佩戴成人帽 / 徽章、宣誓、感恩父母环节）与着装要求；家庭版则近升学宴体例，突出“见证成长”。

**5. 同学聚会措辞注意**
- 具名：以“××届××班同学会筹备组 / 筹委会”具名，不用个人名义号令全体。
- 平等原则：“序长不序爵”——不写官职头衔、不炫富、不排名；强调“只有一个身份：同学”。
- 信息要素：聚会日期（可多日）、报到时间地点、活动日程、费用说明（多为 AA 制，写明人均预算）、报名回执方式与截止日期、联系人。
- 措辞基调：怀旧而不伤感，调侃而不轻慢；“别因你的缺席而使聚会失色”是最经典收束句。
