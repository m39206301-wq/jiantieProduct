# 节庆家宴 · 白事讣告 —— 中文请柬文案集（终稿）

## 场景总述

本文集收录两类场景文案，共 **22 则**。

- **传统节日家宴请柬（15 则）**：覆盖春节团年饭 / 年夜饭、中秋、元宵、端午、重阳五大节日。春节重"团圆、守岁、天伦"，宜热烈喜庆；中秋重"团圆、赏月、共婵娟"；元宵重"花灯、汤圆"；端午重"粽香、艾草、安康"（宜说"安康"，不宜只说"快乐"）；重阳重"登高、赏菊、敬老"，邀请对象是长辈时称谓须用尊称。
- **白事讣告 / 讣闻（7 则）**：传统式 3 则、现代式 4 则。总原则：**措辞克制、体面、规范**。语言准确、简练、严肃、郑重，切忌花哨修饰与口语玩笑。一般于遗体告别仪式前数日发出，以便亲友准备花圈、挽联。书写或印刷用白纸黑字；传统上长辈之丧用白纸（或四周加黑框），幼辈之丧可用黄纸（从俗）；电子讣告可用黑底白字。

每则文案统一标注【篇幅】【适用场合】【风格】【适配载体】；正文中 ××× / ×× 为占位符，使用时替换。文案篇幅分档：短＝一至三句，适合微信 / 电子请柬 / 短信；中＝完整请柬正文；长＝含缘由、流程、寄语的完整邀请函。

> 说明：本文件只保留节庆家宴与讣告文案及各自附录；通用请柬格式规范（竖排 / 横排、称谓、敬语、干支纪年等）与单页 / 双开页 / 三折页版式分配内容，另见专门的格式指南文件。

## 附录：写作规范与术语表

### 附一、节庆家宴写作要点（分节日）

**春节团年饭 / 年夜饭**
- 核心意象：辞旧迎新、团圆、守岁、年夜饭、爆竹、春联、天伦。
- 时间写法：除夕多写"农历腊月三十（廿九）晚"或"公历×月×日（星期×）"；家宴请柬宜公历、农历并书。
- 措辞宜热烈喜庆，可用"团年""守岁""阖家团圆""共叙天伦"。
- 现代家庭多由晚辈具名邀请家族成员，具名可用"××全家敬邀"。

**中秋家宴**
- 核心意象：团圆、赏月、月饼、桂花、千里共婵娟。
- 中秋是仅次于春节的"团圆节"，邀请核心词为"团圆""赏月""共婵娟"。

**元宵家宴**
- 核心意象：花灯、汤圆、上元、月圆、灯如昼。
- 可结合"赏花灯、猜灯谜、品汤圆"等活动写。

**端午家宴**
- 核心意象：粽香、艾草、龙舟、安康（端午宜说"安康"，不宜只说"快乐"）。
- 家宴规模一般小于春节中秋，措辞可清新雅淡。

**重阳家宴**
- 核心意象：登高、赏菊、敬老、九九长寿。
- 重阳家宴多以敬老为主题，宜突出"敬老""感恩""陪伴"，邀请对象是长辈时称谓须用尊称。

### 附二、讣告的三种现代形式

1. **一般式**：民间最常用。标题"讣告"（或冠以逝者姓名），正文写逝者姓名、身份、逝世原因、日期、地点、终年岁数；可简介生平；末写吊唁 / 追悼会时间地点；落款为家属（姓名后加"哀告"）或治丧组织，注明日期。结语常写"特此讣告""谨此讣闻"。
2. **公告式**：隆重庄严，用于重要人物，由机关、团体决定发出，如"×××同志逝世公告"。
3. **简便式**：作为一则消息在媒体 / 网络上公布，旨在晓谕社会，文字极简。

### 附三、讣告传统术语体系（必须准确）

**1. 对逝者的称谓**
- 父死曰**考**，母死曰**妣**（"生曰父曰母，死曰考曰妣"）。父丧称"先考"（父有功名、德望或上无长辈则用"显考"）；母丧称"先妣／显妣"。
- 亡父尊称**府君**，亡母尊称**太君／老孺人／太夫人**；名前加"**讳**"，示不敢直呼。
- 祖父丧：显祖考××太府君；祖母丧：显祖妣×太夫人。
- 变通写法（民间简化）："考妣"可写作"严君／慈亲""家严／家慈""先严／先慈"。

**2. 寿数与死亡表述**
- 男死书"**寿终正寝**"，女死书"**寿终内寝**"，年少者用"终于正寝／终于内寝"。对长辈用"痛于／恸于"，对晚辈用"悼于／哀于"。
- 年龄写法（民间惯行，各地略有出入）：60 岁以上曰"**享年**（享寿）"；60 岁以下曰"**存年**"；又有一说：三十曰得年、四十曰纪、五十曰终、六十曰享年、更高寿曰"耈寿／期颐"。三十以下可写"得年／存年"，意外身故可写"殁年"，十岁左右曰"夭折"。
- 距生于×年×月×日，享年×十有×岁（"×十有×"为文言计数，如"享寿七十有二"）。

**3. 关键套语**
- "不孝男××等，罪孽深重，弗（不）自殒灭，祸延显考……"——自谦自责之辞，谓先辈之亡乃己罪所致。
- "随侍在侧，亲视含殓，遵礼成服"——含殓：于逝者口中置物以殓；成服：按亲疏穿应持丧服。
- "停柩在堂""泣卜×月×日家奠／开吊／发引／扶榇安葬"。
- 结尾："叨在戚／友／寅／学／乡／世谊，哀此讣闻"（"谊"字例抬高一格或加大，"闻"字常加大；传统印本"戚友寅学乡世"等字可用红色）。亦作"倘蒙姻世乡族谊垂赐吊唁，曷胜哀感，谨此讣闻"。

**4. 孝眷自称与哀泣词（具名体系，依"五服"排列，不得逾越）**
- 父死母在，子称"**孤子**"；母死父在，称"**哀子**"；父母双亡，父先亡母后亡称"**孤哀子**"，母先亡父后亡称"**哀孤子**"。无父曰孤，无母曰哀。
- 父亡而继母在堂，子自称"孤哀子"并加小注"**奉继慈命称哀**"（或"侍继慈孤哀子"）；生母在则"奉慈命称孤哀"。
- 长子先殁而祖父母丧，长孙领衔主丧，称"**承重孙**"（代父承重，列名第一，虽有叔父在亦居前）。
- 出继为人后，本生父母死，称"**降服子**"；妻亡，夫在堂有父者称"不杖期夫／期服生"，无父者称"杖期夫（杖期生）"；父为子媳发讣，自称"反服父（反服生）拭泪拜"。
- 哀泣词按服制轻重递降：
  - 斩衰三年（子为父母、承重孙）：**泣血稽颡**（泣血：悲痛泣下如血；稽颡：以额触地且停留，服丧三年者行此大礼）。
  - 齐衰期服（孙为祖父母）：**泣稽首**（亦作泣血稽首，各地从俗）。
  - 齐衰五月／三月（曾孙、玄孙）：**抆泪稽首／抆泪顿首**。
  - 大功九月（堂侄等）、小功五月、缌麻三月（侄孙、侄曾孙等）：**拭泪顿首／沾襟顿首**。
  - 女婿：拭泪顿首；孙女、外孙等：泣泪拜、挥泪拜；胞兄弟：顿首；胞姐妹：敛首；其他亲戚：鞠躬。
- 新式讣告落款可用"同泣跪启""同泣启""泣血叩启""泣血谨启""拜告"等。
- 人名过多时，末注"族繁不载"。

### 附四、讣告注意事项（速查）

1. 时间：务必在告别仪式前发出（宜提前 2–3 日以上）。
2. 纸色字体：白纸黑字，字体宜楷、隶，标题大于正文；电子讣告可黑底白字。长辈丧白纸，幼辈丧可用黄纸（从俗）。
3. 称呼不可错：父—考／正寝；母—妣／内寝；孤子、哀子、孤哀子按父母存亡区分；"享年"与"存年／得年"按年龄区分。
4. 称谓从逝者立场写："家父、家母、先严、先慈"为家属口气；单位讣告用"×××同志"。
5. 亲属具名传统式只列男丁、依五服排序；现代式可列子女、儿媳、孙辈，从俗并列女儿女婿。
6. 如有遗嘱（丧事从简、谢绝礼金、遗体捐献等）应予写明。
7. 忌用喜庆字眼与色彩；不使用红色（传统"戚友"等字用红为例外旧俗，今多不用）。
8. 传统文言套语（泣血稽颡、台光、阖第光临等）只在整体文言或传统格式中使用，与现代白话混用时须保持语体统一，避免半文半白出错。
