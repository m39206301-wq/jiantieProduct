# 乔迁之喜·请柬文案（29 则）

> **场景总述**：乔迁帖区别于其他喜帖的核心要素是“新址”——务必写到门牌。本集涵盖四大场景：**家宅**（新居落成／乔迁新居／入伙宴请，11 则）、**乡村自建房**（上梁／落成酒，6 则）、**办公室／公司乔迁**（7 则）、**商铺迁址**（5 则）；风格覆盖传统庄重（含典故意象）、现代简洁、温馨家常三档。篇幅分短（约 100 字内，帖心简式／短信微信）、中（100—250 字，完整请柬正文）、长（含新址、时间、席设、落款的完整全文，可直接套用）三档。帖中“××”为占位符，已填具体日期地址者为范例数据，使用时整体替换。

## 附录：写作规范与常用语库

### 附录一、“乔迁”典源与用词规范

1. **典出《诗经·小雅·伐木》**：“伐木丁丁，鸟鸣嘤嘤。出自幽谷，迁于乔木。”乔，高也。鸟自幽深山谷飞升高树，本谓“从低处迁到高处”，后用于贺人迁居或官职升迁，天然带有“由小到大、由低到高、越来越好”的褒义指向。
2. **同源词族（均可入帖）**：乔迁、迁乔、莺迁、迁莺、出谷迁乔、出幽迁乔、莺迁乔木、乔木莺声、燕徙。（“莺”由“鸟鸣嘤嘤”衍化而来，初多贺登第升擢，后亦贺迁居。）
3. **经典意象（均已核验出处）**：
   - **美轮美奂**——《礼记·檀弓下》，专颂屋宇高大华美，贺新居最正宗的典故之一；勿移用于宴席、服饰等非建筑物；
   - **竹苞松茂**——《诗经·小雅·斯干》（宫室落成之歌），贺新屋落成最雅之词；
   - **燕贺／燕雀相贺**——《淮南子·说林训》“大厦成而燕雀相贺”，衍生“燕贺德邻”“燕入高楼”“莺迁燕贺”；
   - **里仁为美／德必有邻**——《论语·里仁》，贺迁居择得仁里芳邻；
   - **堂构增辉／肯堂肯构**——《尚书·大诰》，贺营建第宅；
   - **金玉满堂**——《老子》九章，贺新居殷实；**紫气东来**——《列仙传》，祥瑞临门，上梁联常用；
   - **良辰安宅，吉日迁居**——民间贺联最高频句式。
4. **误用警示（不可错用）**：
   - **麟趾呈祥、弄璋之喜、梦熊之喜**——皆贺生子，非贺乔迁。尤须注意：弄璋、梦熊与“竹苞松茂”同出《诗经·小雅·斯干》，最易因同篇误植；贺宅只用“竹苞松茂”，生男才用“弄璋、梦熊”。
   - **左迁**——贬官之谓，大忌，与“乔迁”义相反，切勿混用。
   - 迁入条件明显更差的住所或自谦租屋斗室，宜改用“迁居、移居、迁寓”；雅帖自题多谦称“蜗居粗就、寒庐新构、蓬荜初成”。
   - 对外正式请柬宜用“乔迁之喜／新居落成”；面向乡邻亲友的口头邀请、电子帖可直书“温锅”“入伙”“上梁大吉”，亲切得体。

### 附录四、“温锅／暖房”民俗要点

- **概念与别称**：乔迁后主人择吉日设宴，遍邀亲友上门“认新家门、添人气、暖新宅”。此俗北方通称“温锅”，又称温居、暖房；广东称“入伙”；扬州称“进宅”；山东、东北尤盛。民间认为只有“支了锅、温了锅”，乔迁才算圆满。
- **择日**：搬家必择黄道吉日，干支须与家主生辰相合相生；旧俗忌闰月搬家。请柬宜公历＋农历＋星期并书，便于长辈核看“好日子”；上梁、入伙另须注明吉时。
- **威海“支锅—温锅”流程**（威海市档案馆《威海老风俗》）：首次生火开灶称“支锅”——新锅先蒸发糕（“支锅糕”，取“高”意），再煮水饺（“小元宝”），先敬灶王，然后家人方可食用。温锅日妻族亲戚必带发面大饼与面元宝（大饼染粉红色者寓“日子红火”，蒸熟顶破皮为吉兆）；每位亲戚必带一个盆和两把筷子——“送聚宝盆”、祝其“快发家”。回礼将大饼切块分给众亲戚与左邻右舍，取“大家一起发”；筷子主客各留一把，俗谓“你一把，我一把，两家伴着发”。
- **各地旧俗摘记**：山东沂源温锅礼带柴米油盐酒菜，讲究人家带大公鸡与大鲤鱼（大吉大利、年年有余、鲤鱼跳龙门）；杭州、绍兴搬入次序为先“发蓝”（取“发达”）、再梯子与小儿衣裤（“节节高”）、再吉祥草万年青与柴米、最后神堂祖先堂，他人不得言语，谓之“闷声发大财”；扬州进宅先搬吉祥草、万年青，再放鞭炮迎家堂神龛，拎进“子孙桶”，家具上放芝麻秸、安豆、芦柴（“节节高”“安安稳稳”“长远发财”）；广东入伙备新火柴一盒、八分满清水桶（内置双数碗筷）、米缸装米八分满以上并贴红纸“常满”，吉时到主人捧火盆绕新居一周，入宅者不可空手，礼成后开灶煮汤圆、泡茶飨客。
- **与请柬文案的结合点**：① 正式请柬事由写“乔迁之喜／新居落成”，至亲乡邻的电子帖、口头邀请可直书“×月×日温锅（暖房／入伙），恭候光临”；② 辞谢厚礼可加注“人来就是贺，仪礼从简／坚辞厚贶”；③ 上梁帖可与对联“上梁欣逢黄道日，立柱巧遇紫微星”呼应；祭梁吉语（“祭梁头，儿孙代代出公侯……”）可作酒宴助兴词，不宜印入帖面。

### 附录五、写作要点与避坑清单

**必须做到**

1. 新址写到门牌（乔迁帖核心要素）；设宴者“席设”写到酒店楼层厅名。
2. 公历、农历、星期并书；“恭候／入席”两个时间分开，恭候宜提前约半小时。
3. 落款夫妇联名（男右女左）或单位全称＋“敬约／谨邀／敬邀”＋发帖日期。
4. 称谓用全称，敬辞成套（送呈—台启—谨订—敬备薄酌—恭请光临—敬约）。
5. 电子请柬附定位、停车指引、联系电话、回执。

**常见错误**

1. 以“麟趾呈祥”“弄璋之喜”贺乔迁（均为贺生子之典）。
2. 误写“左迁”（贬官）；迁居条件更差者用“乔迁”亦失当。
3. 帖面漏写农历或星期，长辈宾客不便核日。
4. 公司帖只发“迁址通知”而无邀请语（乔迁庆典帖＝通知＋致谢＋邀约三层齐备）。
5. 落款只写日期不写姓名，或姓名用昵称缩写。
6. 用典堆砌失当：宅邸用“美轮美奂、竹苞松茂”，德邻用“里仁为美、燕贺德邻”，公司用“鸿猷丕展、骏业日新”——各归其位，勿混搭生子、祝寿之词。
