# 00 请柬格式与载体指南

> 本指南为「中式请柬经典文案库」的纲领性文件，整理通用请柬的结构、称谓、敬语、具名与日期写法，并详解**单页 / 双开页 / 三折页**三种载体的版式与文案分配方法。
> 使用建议：先读本指南掌握规范，再按「场景 → 风格 → 篇幅 → 载体」到 01–08 各分册选用文案；各分册每则文案均标注【短／中／长】档与【适用场合】【风格】，与本指南第五章的载体分配规则配套使用。
> 文中 ××× / ×× 为占位符，实际使用时替换为真实姓名、日期、地点。

## 一、请柬通用结构（五部分）

### 1.1 五部分结构速查

| 序 | 部分 | 内容与要求 |
|---|---|---|
| 1 | **标题** | 封面书"请柬（请帖）"二字，字体大于正文，可做艺术加工（烫金、名家书法、图案装饰）；单页式无封面者，标题写于首行顶端 |
| 2 | **称呼** | 顶格写被邀请者姓名／单位名称，后加冒号（竖排则于首行顶格）；姓名宜用全名，**不得用乳名、昵称、缩写** |
| 3 | **正文** | 写明事由、时间（公历＋农历，附星期）、地点；若有节目、流程、着装、回执等要求一并注明 |
| 4 | **结尾（具礼）** | 礼节性恭候语，如"敬请光临""恭候台光""此致敬礼"，古称"具礼" |
| 5 | **落款** | 邀请者（单位或个人）名称＋敬语（敬邀／谨邀／鞠躬等）＋发柬日期；全家出面可署"×××携全家敬邀"，以长辈为先 |

### 1.2 核心五要素（缺一即废）

| 要素 | 内容 | 常见位置 |
|---|---|---|
| 何人 | 受柬人（称呼）＋具名人（落款） | 首行／帖尾 |
| 何事 | 事由（为××举行××典礼／宴会） | 正文起句 |
| 何时 | 公历＋农历＋星期＋时刻 | 正文 |
| 何地 | 席设详址（区、路、号、厅名） | 正文末或单列"席设："一行 |
| 敬请语 | 恭候台光／敬请光临等 | 结尾 |

> 核对口诀：**先定场合色调**（喜：红金热烈；寿：红金敬祝；节庆家宴：暖、团圆；白事：白黑克制），再逐项核对五要素。

### 1.3 竖排传统格式

- 书写方向：**自右而左、自上而下**。正文中新人／主宾姓名、对对方的敬称（如"台光""光临"）常**换行顶格或抬头**（抬高一格）以示尊敬。
- **男右女左**：夫妇联名时男名在右（先写）、女名在左。
- 竖版帖日期用**中文数字**（一、二、三……或大写壹、贰、叁），不用阿拉伯数字。
- 传统骨架（婚宴、寿宴、家宴通用）：

> 送呈　×××先生　台启
>
> 谨订于×年×月×日（星期×）×时　（农历×月×日）为×××举行××之喜　敬备薄酌
>
> 恭候
> 台光
>
> ×××　敬邀
> 席设：××酒楼××厅

- 极简老式帖（民间通行格式）：

> ×月×日，小儿完婚。敬备喜筵，恭候
> 台光
> 眷弟×××拜

- 发柬时机：《东京梦华录》"凡宴集，先三日具帖"——传统上请帖提前约三日送达；今婚喜大事宜提前 7–15 日，外地亲友再提前。

### 1.4 横排现代格式

- 自左而右、自上而下，同现代书信；称呼顶格加冒号，正文空两格起；横排可用阿拉伯数字，正式帖仍建议中文数字。
- 标准模板：

> ×××先生／女士：
>
> 谨定于××××年××月××日（星期×）××时（农历×月×日），在××酒店××厅为×××举行××宴会，敬备喜筵（薄酌）。
>
> 恭请
> 光临
>
> ×××、×××　敬邀
> ××××年××月××日
> 席设：××路××号××酒店××厅

- 现代简式（双联署名）：

> ×××先生／女士：
> 我俩定于×月×日×时在××举行结婚典礼，并设筵招待。恭候
> 光临
> ×××、×××　鞠躬
> 席设：××酒店　时间：×时入席

## 二、称谓表

### 2.1 封面／信封启封词（"启"字系列，按身份选用）

| 对象 | 启封词 | 说明 |
|---|---|---|
| 尊长、领导 | 钧启、赐启、勋启、安启 | 最敬 |
| 师长 | 道启、钧启、安启 | |
| 亲属长辈 | 福启、安启 | |
| 平辈及一般人 | 台启、大启、惠启、台展 | 最常用 |
| 女性（未婚） | 芳启 | 姓名＋小姐芳启 |
| 晚辈 | 收启 | |

> 通行格式："**送呈 ××× 台启**"——"送呈"指恭敬递上，"台启"为启封敬词；**"台启"之后不再写字**。信封竖写时名字写大、称呼略小，更为美观得体。

### 2.2 称呼写法细则

| 邀请对象 | 写法 |
|---|---|
| 个人（男） | 姓名＋先生＋台启／大启 |
| 个人（女，已婚） | 姓名＋女士＋台启；（未婚）姓名＋小姐＋芳启 |
| 夫妻二人 | 姓名＋贤伉俪＋台启／钧启；或"××先生暨夫人" |
| 全家 | 姓名＋全家福（后不再加"收／启"）；信封可书"阖府统请" |
| 有职务者 | 姓＋职务＋名＋启封词，如"黄正德主任大启"；亦可姓名＋职务＋钧启 |
| 姑姨等女性长辈（已婚） | 以姑丈／姨丈之名为主帖对象，不可只写姑姑、阿姨之名 |
| 姑伯叔姨（父母健在） | 姓名＋全家福（视为父母平辈，由父母名义请） |
| 姑伯叔姨（父母已故） | ××姑父大人／伯父大人／舅父大人＋钧启 |
| 单位／团体 | 单位全称＋负责同志　钧启／台启 |
| 亲戚长辈 | 可只写称谓不写姓名，如"尊姑母×××" |

### 2.3 "全福／双福／双恋"用法

| 附注 | 含义 | 写法 |
|---|---|---|
| 全福（全家福／合福） | 请对方**全家老小**出席 | ××先生　全福 |
| 双福 | 只请**夫妻二人** | ××先生　双福 |
| 双恋 | 请**未婚恋人双方** | ××先生　双恋 |

> 只请一人：写全名＋台启（男）／芳启（女）即可，**切勿画蛇添足加"全福"**；请夫妻勿漏"双福"而只写一人名。

### 2.4 正文敬称与自谦称

- **指代对方的敬称**：尊长用"××大人""尊翁""×公"；平辈用"××兄""吾友""台端"；晚辈用"××贤侄（甥）""××同学／同事"。
- **家属自谦称**：家严（父）、家慈（母）、家兄、舍弟、小儿（犬子）、小女；自称住所"寒舍"；借场地设席用"**假座**××酒家"（"假"＝借）。
- 父母具名帖中儿子排行：**长男、次男、三男、四男、幼男**，独子写"小儿"；儿女排行分开计算。嫁女帖写"小女／爱女"，古称"于归／出阁"。

## 三、敬语体系（释义与用法）

### 3.1 起首／约期类

| 敬语 | 释义与用法 |
|---|---|
| **谨订／谨定** | 恭敬地订定（日期）。最通用，如"谨订于×月×日"；二字通用，同一帖内须统一 |
| **谨具** | 恭敬地备办（请柬、筵席），如"谨具喜筵" |
| **敬约** | 恭敬地邀约，常置于末尾"××敬约"或"恭候　敬约" |
| **敬备薄酌／敬备喜筵／敬备菲酌／薄酒／喜酌** | 自谦酒宴微薄；喜事用"喜筵／喜酌"，丧事后酬宾用"薄筵／素筵" |
| **淡酌候教／喜酌候教** | 简宴恭候指教（旧式出阁帖常用"荷蒙厚仪，谨订于是日×时淡酌候教"） |
| **假座××** | "假"＝借，借某处设席的谦辞，如"假座××酒家×楼×厅" |

### 3.2 恭请类

| 敬语 | 释义与用法 |
|---|---|
| **恭候** | 恭敬地等候，后接"台光／光临／驾临" |
| **恭请／敬请** | 恭敬地邀请；"恭请光临"前后都不再写字 |
| **台光** | "台"为敬辞（如台甫、台鉴），"台光"即尊驾之光；传统竖排中"台光"多换行抬头，港澳及老式帖常用 |
| **光临／驾临／贲临** | 客人到来之敬称 |
| **阖第光临／阖府光临** | 请对方全家到来（"阖第"即全家），用于请全家赴宴 |
| **移玉／移贵趾** | 请人动身前来之婉辞，如"望百忙中移玉" |
| **随候尊驾** | 随时恭候大驾，半文半白帖常用 |

### 3.3 具名收尾类

| 敬语 | 释义与用法 |
|---|---|
| **敬邀／谨邀** | 恭敬邀请，用于落款"×××敬邀"，现代通行 |
| **鞠躬／顿首／拜／敬叩／肃拜** | 具名后敬语，按辈分选用：平辈用"鞠躬／顿首"，对长辈用"拜／顿首拜／敬叩"，对晚辈用"请／恭请"，女主人可用"肃拜／端肃拜" |
| **谨启／拜** | 书信式收尾，"×××谨启""眷弟×××拜" |
| **恕邀／恕不介催／恕乏价催** | 旧式谦辞："恕邀"意为"柬帖之外恕不再一一面邀"；"恕不介催"意为不再派人催促，请自行赴宴（今多慎用，以免误解为简慢） |
| **全福／全家福、双福、双恋** | 帖尾附注请全家、夫妻、未婚情侣同来（见 2.3） |
| **忝眷／眷弟** | 姻亲间的谦称具名，如"眷弟××拜" |
| **承庆／承庆子** | 寿柬中子女为父母具名的自称，帖末署"承庆 ××顿首"；祖父母在世时为父母做寿则称"重庆子" |

### 3.4 其他常见套语

| 套语 | 含义与用法 |
|---|---|
| 送呈……台启 | 恭敬递呈，请您启封（封面／首行）；"台启"后不写字 |
| 席设 | 单列一行注明设席地点 |
| ×时恭候，×时入席 | 迎客时刻／开席时刻，固定搭配 |
| 蓬荜生辉 | 谦称贵客到来使寒舍增光 |
| 荷蒙厚仪 | 承蒙厚礼（谢帖、出阁帖、订婚答谢帖用） |
| 吉时／入席时间 | 注明"恭候×时　入席×时" |
| 合府统请 | 邀请对方全家的旧式信封写法 |
| 于归／出阁 | 女子出嫁（嫁女帖用语） |
| 暨／和 | 连接两个姓名的专用字，**不用顿号、逗号** |

## 四、具名规则与日期写法

### 4.1 具名规则

1. **以谁名义具名**：喜事行帖有四种通行做法——以家长（父母）名义；以兄嫂或推选的主事人行帖；以至亲中有名望的长辈行帖；以当事人（如新人）具名。
2. **以长辈为先**：父母为子女婚事具名写"×××、×××（父母姓名）敬邀"；多人具名并排，尊长居首；家庭聚会可署"×××携全家敬邀"。
3. **新人具名**：新郎名在前（或左／右，依竖排男右女左），新娘名在后；两姓名并排竖写表示平等，中间用"暨"或"和"连接。
4. **排序总原则**：血缘亲者在前、长者在前。子女为父母寿宴具名以**长子领衔**，儿媳随夫并列；有女婿共同操办的列"女儿××、女婿××"（女儿在前，血缘为主）。电子请柬群发可统写"××全家敬邀"不分列。
5. **名不可用缩写、昵称、乳名**；写全名。
6. **具名位置（竖式帖）**：约邀人具名须在帖的一半之后下笔，具款末后一字写到帖脚（底），否则视为不敬。
7. 喜事落款可加"鞠躬／谨订／敬约"；白事具名依"五服"体系（孤子、哀子、孤哀子、承重孙等），详见 08 分册讣告部分。

### 4.2 日期写法（农历公历并用）

- **规范做法：公历与农历并书**，如"公历××××年×月×日（星期×）　农历×月×日"；传统帖常上行公历下行农历（竖排则右公历左农历），也可并排竖写，或划去其一只填一种。
- **公历**：可用阿拉伯数字"2025年10月1日"；郑重帖用汉字"二〇二五年十月一日"。
- **农历**：一律用汉字数字。**正月不称一月，十一月称冬月，十二月称腊月**；二十写作"**廿**"，三十写作"**卅**"；初一至初十加"初"字；闰月注明"闰×月"。
- **星期务必标注**，方便宾客："（星期日）"。

### 4.3 干支纪年

- 天干（甲乙丙丁戊己庚辛壬癸）配地支（子丑寅卯辰巳午未申酉戌亥），60 年一周期。
- 雅正写法如"**岁次乙巳**""癸卯年仲秋"；完整农历日期如"乙巳年八月十五日"。
- 注意：干支年以**立春**（一说正月初一）为界，跨年前后书写须查万年历核实，不可想当然。

### 4.4 时刻写法

- 现代写"×时×分（上午／下午）"。
- 仿古写法可用时辰（午时＝11–13 时）并括注现代时刻，如"午时（上午十一时）"。
- 宴席通行"×时恭候，×时入席"：恭候时＝迎客，入席时＝正式开席。

### 4.5 双历并书范例

> 谨定于公历二〇二五年十月一日（星期三）　岁次乙巳年八月初十日　中午十二时　敬备喜筵

## 五、三种载体版式详解与文案分配（重点）

> **总原则**：封面只放标识与受柬人信息；正文核心五要素（何人、何事、何时、何地、敬请语）集中在最易阅读的内页主版面；辅助信息（流程、回执、交通、落款日期）放次版面或封底。记住层级口诀：**封面标识 → 主版五要素 → 副版流程回执 → 封底落款吉语**，信息不重复、不越级。

### 5.1 单页请柬（单帖／单卡）

**版式特点**：一张纸（或一张卡片）承载全部内容，无折页；分竖排单帖与横排单卡。信息必须高度凝练；成本低、派发方便，也是电子请柬的单屏版原型。

**版面分配**

| 位置 | 内容 |
|---|---|
| 顶部 | 标题"请柬"二字（大于正文，可烫金／书法体）；单帖无封面者标题居首行 |
| 首行顶格 | 受柬人称呼："送呈×××台启"（竖排）或"×××先生／女士："（横排） |
| 中部 | 正文——事由、谨定日期（公历＋农历＋星期）、时间、地点、"恭候台光／光临" |
| 下部 | 落款"×××敬邀"＋发柬日期；"席设：××"一行（也可并入正文） |

> 讣告类单页：标题"讣告"居中加大，正文直陈，四周可加黑框。

**文案容量**：正文宜控制在 **80–150 字**；只保留核心五要素，删去流程细节。**适合短档文案，或将中档文案压缩后使用。**

**示例 1｜竖排单帖 · 春节年夜饭（短档全文入帖）**

- 顶部：烫金"请柬"二字
- 首行顶格：送呈 ××× 先生 台启
- 中部正文（引自 08 分册·节庆家宴短档）：

> 爆竹辞旧岁，围炉话团圆。谨定于除夕之夜略备年饭，恭候阖第光临，共叙天伦。

- 下部：席设：××路××号本宅　／　××× 敬邀　／　××××年×月×日

**示例 2｜横排单卡 · 中秋家宴（短档＋信息行）**

- 顶部：艺术字"请柬"配圆月桂花纹样
- 称呼：××× 先生／女士：
- 中部正文（引自 08 分册·节庆家宴短档）：

> 月圆人团圆。中秋之夜，略备薄酌，恭候台光，同赏明月。

- 下部信息行：时间：农历八月十五日晚×时　／　地点：××路××号　／　××× 敬邀

**示例 3｜横排单卡 · 酒店婚宴（中档压缩式）**

- 顶部："请柬"二字（或新人姓名＋"囍"）
- 称呼：××× 先生／女士：
- 中部正文（引自 01 分册·酒店正式中档，全国通行度最高的模板之一）：

> 谨于公历×年×月×日假座××酒家×楼×厅举行婚礼，×时恭候，×时入席。×××、×××敬备薄酌，恭候光临。

- 下部：×××、××× 敬邀　／　联系电话：×××××××

### 5.2 双开页请柬（双帖／对折卡）

**版式特点**：一张纸对折成四面（封面、内左、内右、封底），即传统"双帖"，是中式喜帖最经典的形制：封面显尊重，内页容纳完整帖文，封底收束。传统竖排帖展开后自右向左阅读，**内页右页相当于"帖首"**。

**版面分配**

| 版面 | 内容 |
|---|---|
| **封面** | 居中大字"请柬／囍"（喜事）或"讣告"（丧事用素色、黑框，不用喜庆装饰）；可饰纹样（喜用龙凤／双喜／花团，寿用松鹤，丧用素莲或不饰）；封面下方或封套写"送呈×××台启"（若用封套，受柬人写封套） |
| **内页右（主版）** | 帖文主体——称呼、谨定日期、事由、恭候语，即第一、二章的完整骨架（**中／长正文放这里**） |
| **内页左（副版）** | 补充信息——"席设"详址、恭候与入席时间、行车路线、着装提示、回执方式（RSVP）、联系人电话；喜帖可放新人合照或家训、诗句点缀 |
| **封底** | 落款具名（双方父母／全家姓名）、发柬日期；可加一行吉语（喜："百年好合"；寿："福如东海"；家宴："阖家团圆"）；白事封底素净，可书"哀此讣闻"或不置文字 |

**文案容量**：内页正文 **150–300 字**，副版信息点列式书写。**适合中档文案为主体、短文案作封面主题句。**

**示例 1｜婚宴 · 传统竖排双帖（新人具名）**

- 封面：居中烫金"囍"字＋龙凤纹样；封套书"送呈 ××× 先生 台启"
- 内页右·主版（引自 01 分册·经典"送呈台启"全文）：

> 送呈 ××× 先生（女士） 台启
>
> 谨订于×年公历×月×日（星期×）
> 农历×月×日
> 为××先生和××小姐举行结婚典礼，敬备喜筵
> 恭请
> 光临

- 内页左·副版：席设：××酒店××厅　／　时间：×月×日×午×时　／　×时恭候，×时入席　／　联系电话：×××××××
- 封底：×××、××× 敬邀　／　××××年×月×日　／　吉语"百年好合"

**示例 2｜除夕团年饭 · 横排对折卡（父母／晚辈具名）**

- 封面："团年"主题字＋一句主题语"爆竹辞旧岁，围炉话团圆"＋红灯笼纹样
- 内页右·主版（引自 08 分册·节庆家宴中档传统横排格式）：

> 送呈 ××× 先生（女士）台启
>
> 谨定于公历××××年×月×日（星期×）晚×时（农历腊月××日），在本宅举行团年晚宴，敬备薄酌，共叙天伦。
>
> 恭候
> 台光
>
> ××× 敬邀

- 内页左·副版：席设：××路××号本宅（××酒店××厅）　／　行车路线与停车提示　／　请于×月×日前回复（电话：×××××××）
- 封底：××× 携全家 敬邀　／　××××年×月×日　／　吉语"阖家团圆"

**示例 3｜寿宴 · 传统双帖（子女具名）**

- 封面：金"寿"字＋松鹤纹样；眉首配万能寿联"福如东海长流水，寿比南山不老松"
- 内页右·主版：

> 送呈 ××× 台启
>
> 谨订于×年×月×日（星期×，农历×月×日）中午×时
> 为家严举行八十寿辰，敬备寿筵
> 恭候
> 台光

- 内页左·副版：席设：××酒店×楼×厅　／　恭候×时　入席×时　／　附注：（谨遵父命，不受贺仪）
- 封底：承庆 ××× 顿首　／　××××年×月×日　／　寿词横批"松鹤延年"

### 5.3 三折页请柬（六面折卡／风琴折）

**版式特点**：一张纸两折成六面（外三：封面、折入页、封底；内三：连续内页），容量最大，适合需要承载流程表、地图、回执联、家族信息或图文混排的场合：大家族团年宴、寿宴、纪念性活动、企业年会请柬。现代横排多为左起风琴折；传统题材可做右起经折装。

**版面分配（以左起横排为例）**

| 面 | 名称 | 内容 |
|---|---|---|
| 第 1 面 | 封面 | 标题＋主题视觉＋一句主题语（如"××家族·乙巳团年宴"）；受柬人称谓若不用封套，可置于封面下缘"敬呈×××台启" |
| 第 2 面 | 折入页／扉页 | 欢迎词或缘起短文——为何设宴、致宾客的寄语（100–200 字）；或放家训、诗句、全家福照片，起情感铺垫作用（**长文案的故事段放这里**） |
| 第 3–4 面 | 内页主版 | 正式请柬正文（谨定日期、事由、恭候语）＋活动流程表（签到—开席—余兴—送客）；主宾信息、席设详址在此 |
| 第 5 面 | 内页副版 | 实用信息区——交通与停车、地图二维码、住宿建议、着装要求、RSVP 回执（可设计为可撕回执联）、联系人 |
| 第 6 面 | 封底 | 落款具名全衔、发柬日期、吉语／谢语、主办方标识；白事则保持素净，只具名与日期 |

**文案容量**：全折 **400–800 字**；注意"**封面一句、扉页一段、内页一柬、副页一表、封底一名**"的层级，避免信息重复堆砌。**适合长档文案整体使用，或"短（封面）＋长（扉页＋内页）"组合。**

**示例 1｜×氏家族团年晚宴（长档文案的完整分配，引自 08 分册·节庆家宴长档）**

- 第 1 面·封面：主题语"××家族·乙巳团年宴"＋"同守此岁，共话团圆"
- 第 2 面·折入页（缘起短文，取长档开篇段）：

> 光阴似箭，岁月如梭。自上次阖家欢聚，转眼又是一年。为延续我家除夕团年的老传统，增进骨肉亲情……

- 第 3–4 面·内页主版：正文"谨定于公历××××年×月×日（星期×，农历腊月××日）晚×时，在××酒楼××厅举行×氏家族团年晚宴……欢迎携家人一同出席"＋流程表（签到→"家风传承"小仪：长辈讲述家事、晚辈拜年敬酒→开席→送客）
- 第 5 面·内页副版：回执"为便安排席位，请于×月×日前回复（电话：×××××××）；如不能莅临，亦请提前告知"＋交通停车信息
- 第 6 面·封底：××× 携全家 敬邀　／　××××年×月×日　／　吉语"阖家团圆·共叙天伦"

**示例 2｜婚礼三折页（短＋长组合，图文混排）**

- 第 1 面·封面：新人姓名＋短档主题句（如古风短句"赤绳系足，珠璧联辉"，或现代短句"我们结婚啦"）＋合照
- 第 2 面·折入页：故事感第一人称短文 100–200 字（取自 01 分册现代文艺长档的故事段：相识缘起、致谢宾客）
- 第 3–4 面·内页主版：正式帖文（"谨订于×年公历×月×日（星期×），农历×月×日，为××先生和××小姐举行结婚典礼，敬备喜筵。恭请光临"）＋婚礼流程表（迎宾→典礼→婚宴→敬酒→送客）
- 第 5 面·内页副版：酒店地图与二维码、停车指引、着装建议、可撕 RSVP 回执联
- 第 6 面·封底：双方父母暨新人具名全衔　／　××××年×月×日　／　吉语"百年好合·永结同心"

**示例 3｜中秋家族家宴三折页（引自 08 分册·节庆家宴长档）**

- 第 1 面·封面："人月两圆"主题语＋"××家族·××××年中秋家宴"
- 第 2 面·折入页：寄语段"中秋是团圆的佳节，也是我们家族一年一度的盛会……"
- 第 3–4 面·内页主版：正文"特选农历八月十五日晚×时，在××（地点）设宴，与亲朋好友共赏明月、共度佳节"＋余兴安排（赏月、灯谜与家史回望）
- 第 5 面·内页副版："敬请提前安排时间，携家人莅临；请于×月×日前回复确认"＋联系人：×××　电话：×××××××
- 第 6 面·封底：××家族 敬邀　／　恭祝中秋快乐，人月两圆

### 5.4 三种载体选用速查

| 维度 | 单页 | 双开页 | 三折页 |
|---|---|---|---|
| 信息量 | 小（≤150字） | 中（150–300字） | 大（400–800字） |
| 庄重感 | 中 | 高（传统正体） | 高（隆重／纪念） |
| 适用 | 家宴、普通宴请、讣告 | 婚喜寿宴正帖、节庆家宴 | 家族大宴、盛典、含流程回执者 |
| 受柬人称谓位置 | 首行顶格 | 封套或封面 | 封套或封面下缘 |
| 封底内容 | 无 | 落款＋吉语 | 落款全衔＋吉语／标识 |
| 适配篇幅档 | 短（或压缩后的中） | 中为主＋短封面句 | 长为主，或短＋长组合 |

## 六、短／中／长三档文案的定义与选用建议

### 6.1 三档定义

| 档位 | 定义 | 篇幅 | 典型形态 |
|---|---|---|---|
| **【短】** | 一至三句，主题句／核心敬语 | 约 10–60 字 | 主题句、吉语、封面语、短信、微信群发、电子请柬单屏 |
| **【中】** | 完整请柬正文，五要素齐全 | 约 60–200 字 | 标准请柬正文（称呼＋谨定于＋恭候＋落款＋席设） |
| **【长】** | 含缘由、流程、寄语的完整邀请函 | 约 200–600 字 | 书信式全文（称呼＋缘起＋正文＋流程＋回执＋祝颂＋落款） |

### 6.2 选用建议

1. **按渠道选档**：微信／短信／朋友圈用短档或中档；纸质请柬正文用中档；家族盛典、周年纪念、需交代流程与回执的场合用长档。
2. **按载体选档**：单页→短；双开页→中（封面配短句）；三折页→长（封面配短句）。详见 5.4 速查表。
3. **按对象选档**：尊长、正式往来用中／长档规范式；平辈好友、年轻群体可用短档口语式。
4. **组合使用**：三档并非互斥——"封面一句（短）＋内页一柬（中）＋扉页一段（长）"正是双开页与三折页的标准做法。
5. **语体统一**：文言套语（台光、阖第光临、敬备喜筵等）只在整体文言或传统格式中使用；与现代白话混用时须保持语体统一，避免半文半白出错。

### 6.3 篇幅 × 载体 × 场景适配矩阵

| 场景 | 推荐载体 | 推荐篇幅 |
|---|---|---|
| 微信邀请亲友家宴 | 单页（电子单屏） | 短 |
| 婚宴正帖（纸质） | 双开页 | 中＋短封面句 |
| 寿宴正帖（纸质） | 双开页 | 中＋短封面寿联 |
| 节庆家族大宴（含流程回执） | 三折页 | 长 |
| 婚礼图文请柬（含故事、地图） | 三折页 | 长＋短封面句 |
| 讣告 | 单页（或电子单屏） | 短／中（克制直陈） |
| 商务开业、年会 | 双开页／三折页 | 中／长 |

## 七、通用写作禁忌速查（传统用语易错点）

### 7.1 称谓与启封之忌

- 受邀人写**小名、昵称、缩写**——必须用全名。
- "台启""敬邀"之后**续写文字**——启封词、敬语收尾后不再加字。
- 只请一人却写"全福"；请夫妻忘写"双福"而只写一人名。
- 请已婚女性长辈（姑、姨）只写其本人名字——应以姑丈／姨丈之名为主帖对象。
- 两个姓名之间用**顿号或逗号**——应用"暨／和"连接。

### 7.2 敬语套语之忌

- "恭请光临"前后加字——此为独立敬语，前后都不再写字。
- "敬备喜筵"误作"敬备喜宴"混入古文句式（现代帖"喜宴"可用，但与"恭候台光"等老式敬语搭配时宜用"喜筵／喜酌"）。
- "恕邀／恕不介催"等旧式谦辞今多慎用，以免误解为简慢。
- 喜事帖误用丧事后酬宾的"薄筵／素筵"；白事帖误用喜庆字眼与红色。

### 7.3 具名与日期之忌

- 竖版帖用阿拉伯数字写日期——竖版一律用中文数字。
- 具名末字未写到帖脚（竖式帖）——视为不敬。
- 农历用阿拉伯数字；正月写成"一月"；漏写"廿／卅／初／闰"；忘标星期。
- 干支纪年不查万年历，跨立春（或正月）边界写错干支。
- 寿宴日期写在寿辰之后——旧俗"提前不推后"；"做九不做十"等地域差异以当事老人家乡习俗为准，撰稿不可自作主张。

### 7.4 语体之忌

- 半文半白：传统文言套语（泣血稽颡、台光、阖第光临等）只在整体文言或传统格式中使用；白话帖不夹生文言。
- 场合色调错乱：喜（红金热烈）、寿（红金敬祝）、节庆家宴（暖、团圆）、白事（白黑克制）——色调决定措辞与敬语选择，不可混用。

### 7.5 白事三查（务必核对）

1. **查考妣正内寝**：父—考／寿终正寝；母—妣／寿终内寝，切勿互误。
2. **查孤哀子名分**：父死母在称"孤子"，母死父在称"哀子"，父母双亡按先后称"孤哀子／哀孤子"。
3. **查享年／存年**：60 岁以上曰"享年（享寿）"，60 岁以下曰"存年／得年"；措辞宁简毋滥。
