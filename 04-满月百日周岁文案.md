## 附录：写作规范与民俗禁忌

### 一、三个节点与传统用语对照

- **满月酒（弥月之喜）**：出生满一个月设宴，“弥月”即满月雅称，男女通用。
- **百日宴（百岁宴）**：出生一百天庆贺，民间称“过百岁”，取“长命百岁”吉意；贺礼常见长命锁、带绒穗衣裤（“穗”谐“岁”）。
- **周岁宴（抓周／晬盘之喜）**：满一周岁行抓周礼——陈印章、笔墨、算盘、钱币等于盘（晬盘），观其先抓何物以卜志趣。抓周又称试儿、试晬、拈周、试周，南北朝《颜氏家训》已有记载，宋《东京梦华录》称“此小儿之盛礼也”。

| 用语 | 含义与用法 | 性别适用 |
|---|---|---|
| **弄璋之喜** | 生男孩。璋为美玉（《诗经·小雅·斯干》“乃生男子……载弄之璋”） | 男宝专用 |
| **弄瓦之喜** | 生女孩。瓦即纺砖，非屋瓦（“乃生女子……载弄之瓦”） | 女宝专用 |
| **弥月之喜** | 婴儿满月 | 男女通用 |
| **汤饼之喜／汤饼会** | 本指出生三日（洗三朝）庆贺宴，今可雅称满月／百日／周岁婴宴，“敬备汤饼”即用此典；不宜以“汤饼之喜”指代生男或生女 | 通用（活用） |
| **晬盘之喜／晬盘之期** | 周岁试儿（抓周）之喜，“晬”音岁 | 周岁通用 |
| **天降麟儿** | 贺人生子，麒麟为祥瑞，常与“弄璋之喜”连用 | 偏男宝 |
| **添丁之喜** | 家中增添人口 | 男女通用 |
| **喜得贵子／喜得千金（掌上明珠）** | 贵子＝男；千金／明珠＝女 | 分性别 |
| **握珠之喜** | 庆祝家中新添人口 | 男女通用 |

**红包题词（包底）**：贺生子三朝写“汤饼之敬”；男满月写“弄璋之喜”；女满月写“弄瓦之喜”；周岁写“晬盘之敬”。金额用双数。

### 二、旧式经典帖式（民国范本）

> 某年某月某日午时，为小儿**弄璋之喜**（或小女**弄瓦之喜**），**敬备喜酌，恭酬光临**。

——旧时大户人家满月散帖的标准写法。男用弄璋、女用弄瓦，一目了然，**绝不可混用**（本册 A13 即用此式）。

### 三、请柬基本结构（四部分）

1. **封面**：标明“满月请柬”“百日宴请帖”“周岁请柬”等字样。
2. **称呼**：顶格写被邀请人全名及称谓，**切忌写别名、绰号**，后加冒号。
3. **正文**：另起一行空两格，写清事由（弥月／百日／周岁之喜）、时间、地点。
4. **结尾具礼＋落款**：敬语（恭请光临／敬邀／谨启）＋发帖人姓名＋日期，必要时附席设、路线、回执方式。

**传统帖式逐行拆解**：

```
送呈 ×××先生（女士）台启        ← 受邀人姓名＋尊称
谨订于 ××年×月×日（星期×）      ← 时间，宜公历、农历、星期俱全
农历×月×日 ×时
为吾儿（吾女）×× 庆弥月之喜     ← 事由＋宝宝名；百日则“百日之喜”，周岁则“周岁／晬盘之喜”
敬备薄酌 恭候 光临              ← 具礼
×××、××× 敬邀                  ← 父母具名（夫妇并列）
席设：××酒店××厅                ← 地点务详
×时恭候 ×时入席                 ← 恭候与开席时间分列
```

### 四、关键词用法辨析

- **谨订／谨定**：均可，“谨订于……（日期）”为约定恭定之义，比“兹定于”更庄重。
- **送呈……台启**：“台启”为敬请启封之意，用于平辈；对尊长用“钧启”。邀夫妇写男方名＋“夫妇”；邀全家写主人名＋“全家福”或“阖府统请”（**加“全家福／阖府统请”后不再加“台启”**）。
- **敬备／略备／聊备**：谦辞，自谦所备不丰；可搭配薄酌、薄酒、薄宴、菲酌、小酌、喜酌、喜筵、姜酌（旧时帖中谦称酒馔，今少见但有古味）、汤饼（活用汤饼会典故）。
- **恭候／恭请／敬候　光临**：“恭请携眷光临”用于邀对方带家属。
- **落款敬辞**：敬邀／敬约／谨启／敬上；夫妇共同发帖并列双方姓名＋“敬邀”。
- **×时恭候，×时入席**：恭候时间早于开席约半小时。
- **传统称谓（旧式帖／粤港帖仍用）**：请男长辈“德驾”，平辈“台驾”，晚辈“文驾”；请女长辈“懿驾／慈驾”，女平辈“淑驾”。
- **承……命**：极传统写法，祖父在世时孙辈帖子加“承严命／承慈命”，今多省略。

### 五、传统用语硬性规则（易错点）

1. **弄璋之喜＝生男，弄瓦之喜＝生女，绝不可混用**。女宝帖误用“弄璋”是最常见硬伤。
2. **弥月之喜**＝满月，男女通用；“敬备汤饼”为活用古义，不宜以“汤饼之喜”指代生男生女。
3. **晬盘之喜／晬盘之期**＝周岁（抓周）；贺礼题词用“晬盘之敬”。
4. **谦敬称谓不可错位**：自家儿子谦称“犬子／小儿／家儿”，女儿“小女”；对方子女敬称“令郎／令爱／令孙”，绝不可用于自家。祖辈发帖称“家孙／孙女”，父母发帖称“吾儿／吾女”。
5. **贵子／千金**：男宝“喜得贵子”“天降麟儿”“添丁”；女宝“喜得千金”“掌上明珠”“弄瓦之喜”；“明珠入拿（掌）”为贺女经典语。
6. 旧帖性别用语今作雅称性别标识即可，文案语境宜喜庆平等。

### 六、实操注意事项

- **五要素齐全**：宝宝姓名（大名＋小名）、事由、公历＋农历＋星期、时刻（恭候／入席）、地点（酒店＋厅＋详细地址，可附路线／停车／导航）。
- **父母双方具名**：传统并列夫名在前；现代常并列或“××携妻××”。宝宝名必须出现，被邀请人姓名用全称。
- **发帖宜早**：俗云“三天为请，两天为叫，一天为提”，至少提前3–7天送达；电子请柬可附回执与导航。
- **祖辈发帖**常见于满月／百日（“为家孙××庆弥月之喜，××一家敬邀”），称谓随具名人辈分变化。
- **幽默文案分寸**：“记得红包”“不醉不归”“带一张嘴来”“收份子钱”等调侃只适合同龄密友群；发长辈、领导、正式场合请用庄重体。
- **电子请柬（H5／微信）**：可配宝宝照片、成长相册、背景音乐、地图导航；正文仍以“事由＋时间＋地点＋具名”为骨架，首页放最核心信息。
- **二胎／多胞胎**：点明排行（次子／次女／二宝）或“龙凤胎／双胞胎”字样；龙凤胎并列二名，如“为吾儿××、吾女××举行满月酒席”。
- **抓周礼写入请柬**：周岁宴若有抓周仪式，注明“×时抓周礼，×时入席”，方便宾客观礼；雅称可用“晬盘之喜”。
- **避开不吉联想**：婴宴帖用红色／暖色，不用白纸黑字帖式；红包金额用双数。
