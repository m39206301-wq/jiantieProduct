# 开业庆典·周年庆典·年会答谢·发布品鉴招商·奠基竣工——商务请柬文案（42 则）

> **场景总述**：商务请柬的要义是“简要精练、大方得体、首尾呼应”。本集覆盖五大类商务活动：**店铺／公司开业庆典**（10 则）、**公司周年庆典**（6 则）、**年会／答谢晚宴**（7 则）、**产品发布会／品鉴会／招商会**（12 则）、**奠基／竣工（落成、乔迁）仪式**（7 则），共 42 则（短 13／中 18／长 11）。风格分正式公文、商务礼仪、品牌调性（高端／轻奢）三系。篇幅分短（一至三句，信息直达）、中（称谓＋缘由＋时间地点＋邀请语＋落款的完整正文）、长（含议程、回执或确认方式的完整全文）三档。占位符“××”使用时整体替换；范文中的真实公司名已脱敏。

## 一、店铺／公司开业庆典（10 则）

> 写作要点：开业文案三件事——“我是谁、何时何地开业、为何值得您来”。公文式以“兹定于……举行开业庆典，诚挚邀请您的莅临”为骨架；门店类可加经营定位一句；剪彩仪式须写明仪式时间与嘉宾层次；开业酒会宜强调“答谢、薄酌、恭候”。吉时（10:18、10:28、9:58）与“点火投产”“盛大开业”是开业专属语汇。

## 附录：写作规范与常用语库

### 附录一、商务请柬基本结构（六部分）

1. **标题**：居中写“请柬”“邀请函”，或“××开业庆典邀请函”；封面柬可只写活动名＋“邀请函”。
2. **称谓**：顶格书写。个人用“尊敬的 ××× 先生／女士”；重职务者用“尊敬的 ×× 局长／× 总／×× 董事长”；单位用“××公司：”“贵单位：”；不具名用“尊敬的贵宾”“尊敬的合作伙伴”；传统柬帖用“送呈 ××× 先生（女士）台启”。
3. **正文**：缘由（一句答谢／背景）→ 事项（活动名称、主题）→ 时间（×年×月×日 星期× ×时×分，重要庆典附农历）→ 地点（写到具体楼宇、厅室、路牌）→ 邀请语（诚挚邀请您的莅临）。
4. **结尾敬语**：公文式用“此致 敬礼”（“此致”换行缩两格，“敬礼”再换行顶格）；商业信函用“顺祝 商祺”“顺颂 时祺”；柬帖式用“恭候 台光”“恭请 光临”。
5. **落款**：单位全称（或个人姓名＋职务）＋发出日期；单位具名后可加“敬邀”“谨订”“谨呈”。
6. **附项**：回执表、活动议程、着装要求、路线图、停车提示、联系人／电话／微信、电子请柬二维码。

写作总要求：**简要精练、大方得体、首尾呼应**；时间、地点、联系人分块排列，便于一眼扫读；邀请对象是单位时，写明“诚邀贵单位派代表出席”。

| 个人（通用） | 尊敬的 ××× 先生／女士 | 姓名务必准确，职务不明时用 |
| 有职务者 | 尊敬的 ×× 董事长／× 总／×× 局长／×× 主任 | 商务场合职务优先于“先生／女士” |
| 贵宾不具名 | 尊敬的贵宾／尊敬的来宾 | 群发、电子请柬通用 |
| 合作伙伴 | 尊敬的合作伙伴／尊贵的 ×× 品牌代理商 | 渠道类活动常用“尊贵的……合作商” |
| 单位 | ××公司：／贵单位： | 正文配“诚邀贵单位派代表出席” |
| 群体（典礼） | 各位领导、各位嘉宾，女士们、先生们： | 典礼致辞式邀请函 |
| 传统柬帖 | 送呈 ××× 先生（女士）台启 | 邀全家写“×府 全福”或“阖府统请” |

**常用敬语速查**

| 敬语 | 释义／用法 |
|---|---|
| 谨订于／兹定于 | 正式起笔。“谨订于”谦恭，柬帖多用；“兹定于”公文色彩浓，单位函件多用 |
| 恭请／敬请／诚邀 | 邀请动词三级：恭请最敬，诚邀最常用 |
| 拨冗莅临 | 请对方推开冗务前来。“拨冗”为正写，误作“拔冗”者应避免 |
| 莅临指导／莅临参观 | 用于领导、贵宾、上级单位 |
| 恭候光临／恭候台光 | 结尾等候语。“台光”为敬辞，柬帖体多用 |
| 届时恭候／恭候大驾 | “届时恭候您的光临”为高频套语 |
| 恭迎尊驾 | 极敬，奠基、迎宾类柬用 |
| 如蒙应允，不胜欣喜 | 请求确认出席时常用 |
| 谨此奉邀／特此函达／特此敬邀 | 正式收尾，公文式 |
| 此致 敬礼 | 公文祝颂语，注意换行格式 |
| 顺祝 商祺／顺颂 时祺 | 商业信函祝颂语，“商祺”换行顶格 |
| 敬备薄酌／敬备午宴／略备薄酒 | 自谦设宴 |
| 蓬荜生辉 | 称对方到来使自己增光，门店开业常用 |
| 共襄盛举／共话未来／同叙友谊 | 庆典、答谢类高频收尾语 |
| 敬邀／谨邀／敬约／敬呈 | 落款具名后敬辞，如“××公司 敬邀” |

**易错提醒**

- “拨冗”勿误作“拔冗”；“莅临”是敬辞，只能说对方“莅临”，不可用于己方（“我公司莅临现场”为病句）。
- “此致 敬礼”格式：“此致”紧接正文或换行缩两格，“敬礼”必须另起一行顶格。
- 邀请单位不写“贵公司”于称谓行（称谓行写“××公司：”），正文中才用“贵公司／贵单位”。
- 时间要素要完整：年月日、星期、时分；上午、下午明确，晚间宴会建议写 24 小时制避免歧义。
- “上午10时18分”等吉时写法（10:18、10:28、9:58）是开业庆典常见习俗，可保留。

### 附录三、着装要求（Dress Code）写法

**位置**：正文末尾、“温馨提示”栏或请柬封底，中英文可并列一行。日间仪式（剪彩、奠基）一般要求“正装／商务正装”；晚宴酒会写“鸡尾酒会装／晚礼服”；未注明时默认商务正装。

**中文常用层级**（由隆重到休闲）：礼服（正装礼服）＞ 商务正装 ＞ 商务休闲 ＞ 休闲便装。

**西式 Dress Code 对照**（涉外、高端酒会常用）：

| 英文 | 中文 | 要点 |
|---|---|---|
| White Tie | 大礼服（燕尾服） | 男士燕尾服＋白领结，女士曳地晚礼服 |
| Black Tie | 小礼服（无尾礼服＋黑领结） | 男士塔士多＋黑领结，女士晚礼服 |
| Cocktail Attire | 鸡尾酒会装 | 及膝小礼服／精致套装，酒会答谢宴最常用 |
| Business Formal | 商务正装 | 深色西装套服＋领带 |
| Smart Casual | 精致休闲（商务休闲） | 西裤衬衫／有领上装，忌牛仔、运动装 |
| Casual | 休闲装 | 便装，基本无限制 |

**柬内例句**：

- 着装要求：商务正装（Business Formal）。
- 请着正装出席；男士请系领带，女士请着礼服。
- Dress Code：Black Tie（男士无尾礼服＋黑领结，女士晚礼服）。
- 着装建议：商务休闲（Smart Casual），请勿穿着牛仔裤、运动鞋入场。
- 温馨提示：剪彩（奠基）仪式在户外进行，请着正装并备薄外套。

### 附录四、RSVP 回执写法

RSVP 源自法语 Répondez s'il vous plaît（敬请回复），中文柬内写作“敬请赐复”“敬请回执”“烦请确认出席”。核心要素：**回复截止日期＋回复渠道＋出席人数**，视需要增加：是否带车（停车安排）、到达时间（接送安排）、住宿／接站需求、饮食禁忌。

**柬内例句**：

- 请于×月×日前将回执传真至××××-××××××××，或发送邮件至×××@××.com，以便安排席位与接待事宜。
- RSVP：×××（电话／微信同号），敬请于×月×日前赐复。
- 具体到达时间、出席人数，请予以确认并回复，以便我方安排具体招待事宜。后附回执表一份。
- 活动仅限凭邀请函参加，未获邀请者恕不接待（私密品鉴会、高端酒会常用）。

**回执表基本栏目**：姓名／单位名称／职务／联系电话（手机）／出席人数／是否带车（是□ 否□，用于安排停车）／到达时间／是否需安排住宿／备注（饮食禁忌等）。回执末尾注明回传方式与截止日期：“请务必于××年×月×日前将此回执传真（或微信回复）至×××，以便统一安排。”

### 附录五、中英文对照注意事项

1. **排版**：上中下文对照（中文在上、英文在下）或左右分栏，一份请柬只选一种体系，勿逐句混排；电子请柬建议中、英文各做一个页面／入口。
2. **称谓**：中文柬“姓名＋职务／先生／女士”；英文柬用 Mr.／Ms.＋姓；公司名用注册英文名（×× Co., Ltd.），勿自造缩写。
3. **日期时间**：中文“20××年×月×日（星期×）上午9时”；英文用 “9:00 a.m., Friday, 7 June 20××”。英、美式日期序不同，同一份柬内必须统一；星期、a.m.／p.m. 不要遗漏。
4. **敬语意译不硬译**：“拨冗莅临”→ We would be honored by your presence.；“恭候光临”→ We look forward to welcoming you.；“诚邀您出席”→ You are cordially invited to attend…；“如蒙应允，不胜欣喜”→ We would be delighted by your acceptance.
5. **RSVP 保留原文**即可，写 “Please RSVP by＋日期＋联系方式”；中文页对应写“敬请于×月×日前赐复”。
6. **落款**：中文单位全称在上，英文 “×× Co., Ltd.” 在下；对外正式邀请函建议用带抬头的信纸打印并加盖公章（涉外签证用途尤其必须）。
7. **迷你对照示例**：

> 兹定于20××年6月7日（星期五）上午9时，在××国际酒店宴会厅举行××公司开业庆典，恭请光临。
> You are cordially invited to the Grand Opening Ceremony of ×× Co., Ltd. at 9:00 a.m., Friday, 7 June 20××, at the Ballroom of ×× International Hotel. RSVP: ××× (Tel./WeChat) by 1 June.

### 附录七、风格索引与落地提醒

**按风格索引**

- **正式公文式**：开业01、02、05、09、10；周年03、06；年会02、06、07；发布01、02、03、07、12；奠基01、02、06、07。骨架：“兹定于／谨定于＋时间地点＋诚挚邀请＋此致敬礼／谨此奉邀”。
- **商务礼仪式**：开业03、04、06、07；周年01、02、04；年会01、03；发布05、10；奠基03、05。骨架：“感谢＋邀请＋恭候光临”，温度高于公文式。
- **品牌调性式（高端／轻奢）**：开业08；周年05；年会04、05；发布04、06、08、09、11；奠基04。特征：主题命名（“绽放·超越”“穿越·未来”“品味生活”）、金句开场、专属感措辞（“仅限凭邀请函参加”“恕不接待”“恭迎尊驾”）。

**落地使用提醒**

1. **先发后补**：纸质请柬提前 7—15 天送达；电子请柬、H5 提前 3—7 天，活动前 1—2 天电话或微信二次确认。
2. **长柬结构清单**：称谓 → 缘由 → 活动名称主题 → 时间（含签到）→ 地点（含交通／停车）→ 议程 → 着装要求 → RSVP（截止日＋渠道）→ 落款 → 回执表。
3. **电子请柬**：首页只放主题＋时间地点＋“恭候光临”短句，议程、着装、导航、回执表单放二级页；RSVP 用表单／二维码替代传真。
4. **涉外活动**：中英文分版排版，日期格式统一，敬语意译不直译（见附录五）。
5. **敬语红线**：勿以“莅临”自称己方；勿把“拔冗”当正写；称谓姓名、职务、单位名务必与名片／官网核对。
